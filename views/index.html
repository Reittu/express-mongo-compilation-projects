Full Stack Projects

Anonymous Message Board

  • Create threads and reply to existing threads.

    Also option to report posts and delete them if correct password given.

    /b/

  • Get all boards.

    /api/boards

  • Create a new board.

    Required parameters: name

    /api/threads/<board>

  • Get 10 most recent threads with 3 most recent replies on a specific board.

    /api/threads/<board>

  • Post a new thread.

    Required parameters: text password

    /api/threads/<board>

  • Report a thread.

    Required parameters: thread_id

    /api/threads/<board>

  • Delete a thread.

    Required parameters: thread_id password

    /api/threads/<board>

  • Post a new reply.

    Required parameters: thread_id text password

    /api/replies/<board>

  • Report a reply.

    Required parameters: reply_id

    /api/replies/<board>

  • Delete a reply.

    Required parameters: reply_id password

    /api/replies/<board>

Public Library

  • Add and delete books. Can also post comments on existing books.

    /demos/library

  • Get all posted books.

    /api/books

  • Get a specific book by id.

    /api/books/<book_id>

  • Post a new book.

    Required parameters: title

    /api/books

  • Post a new comment.

    Required parameters: comment

    /api/books/<book_id>

  • Delete a specific book (no restrictions, demo).

    /api/books/<book_id>

  • Delete all books (no restrictions, demo).

    /api/books

Issue Tracker

  • Create and manage issues.

    /issues/<project_name>

  • Get all issues on a specific project.

    Optional filtering query parameters:
    issue_title issue_text created_by assigned_to status_text open

    /api/issues/<project_name>

  • Post a new issue on a specific project.

    Required parameters: issue_title, issue_test, created_by
    Optional parameters: assigned_to, status_text, open

    /api/issues/<project_name>

  • Update an issue.

    Required parameters: id and at least one of the optional parameters.
    Optional parameters: project_name issue_title issue_text created_by assigned_to status_text open

    /api/issues/<project_name>

  • Delete an issue.

    Required parameters: id

    /api/issues/<project_name>

Exercise Tracker

  • Create users and track exercises.

    /demos/exercisetracker

  • /api/exercise/users

  • /api/exercise/log?userId=<userId>from=<from>to=<to>limit=<limit>

    Required parameters: userId

    Optional parameters: from to limit

  • /api/exercise/new-user

    Required parameters: username

  • /api/exercise/add

    Required parameters: userId description duration

    Optional parameters: date

Microservices

Stock Price Checker API

  • Query stock prices of one or two stocks.
    You can also like each stock once per IP.

    /api/stock-prices?stock=<stock_symbol>
    /api/stock-prices?stock=<stock_symbol>&like=true
    /api/stock-prices?stock=<stock_symbol_1>&stock=<stock_symbol_2>
    /api/stock-prices?stock=<stock_symbol_1>&stock=<stock_symbol_2>&like=true

  • Like (both)

Metric-Imperial Converter API

  • Convert metric units to imperial or vice versa.
    Supported abbreviations: "gal", "l", "lb", "lbs", "kg", "mi" and "km".

    /api/convert?input=<value><unit>

URL Shortener API

  • Request a new short URL: /api/shorturl/new/<complete_url>

  • Use the given short URL: /api/shorturl/<short_url>

File Analyse API

  • Upload a file to server: /demos/fileanalyse

  • /api/fileanalyse (multipart/form-data)

    File will be deleted instantly as it's saved since this is public.

Who am I API

  • /api/whoami

Timestamp API

  • Request a date either in millis: /api/timestamp/<millis>

    Or dash separated year, month and date: /api/timestamp/<yyyy-mm-dd>
